#	EXAM

##	1.说明

SSM+Javascrip+bootstrap

##	2.功能

* 用户登录,登录成功后跳转到列表页面
* 主页显示所有电影信息
* 单条删除电影
* 单条新增电影
* 单条编辑电影
* 大小写匹配的模糊查询电影信息
* 分页
* more


##	3.开发环境

* spring
* mybatis
* druid
* MySql
* aspectj
* javascrip
* bootstrap
* more
